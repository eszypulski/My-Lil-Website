What I know about HTML

How I learned 🙉

When I was a middle school kid (ca. 1998), I used this extremely late 90s website called Lissa Explains to customize my Neopets storefront and create incredibly ugly websites with my friends. A decade later, I had to write simple HTML when I worked for an educational website. We had an in-house WYSIWYG editor, but it messed up anything too complicated, so after a while, it was easier to do certain things (nested lists, certain types of image alignment) with code. I rarely use it now, but during the early part of the pandemic, my husband (a high school teacher) suddenly had to suddenly put a lot of content online via various janky LMSs, and I would occasionally edit the code manually when he was having trouble with formatting.

Some simple things I can do 🏋

I can do:

  • Italics
    • Useful for emphasis
    • Also sometimes for titles
    • Or in some novels, dialogue
  • Bold
    • I use this for keywords when I'm writing instructional text. (e.g., Click the Submit button.)
    • I guess it could convey STRONG EMOTION, but that feels a bit much
    • Too many people just do this for headings rather than using styles
  • Underline
    • Not frequently used
    • This is how I learned to do book titles (Moby Dick), but in actual style guides, I feel like I see Moby Dick more often.

And more!
